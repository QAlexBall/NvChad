# Usage
## Setup 
```shell
git clone -b qabv2.0 https://github.com/QAlexBall/NvChad.git ~/.config/nvim
ln n-s ~/.config/nvim/tmux/.tmux.conf ~/.tmux.conf
ln -s ~/.config/nvim/wezterm/.wezterm.lua ~/.wezterm.lua
ln -s ~/.config/nvim/wezterm ~/.config/wezterm
```

## Upgrade
```shell
cd ~/.config/nvim
git pull
```
